Query

Query (ou consulta) refere-se a uma solicitação de informações de um banco de dados. Em bancos de dados relacionais, uma consulta geralmente é escrita em uma linguagem de consulta, como SQL (Structured Query Language), e é usada para recuperar, atualizar ou manipular dados armazenados no banco de dados.

Quarentena

No contexto de tecnologia, existe a quarentena de arquivos, onde se isola um arquivo suspeito, para ser analisado e para ter certeza de que não há malwares escondidos.
