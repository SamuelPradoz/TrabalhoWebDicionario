Java

Linguagem de programação de alto nível e uma plataforma de computação criada pela Sun Microsystems (agora pertencente à Oracle). Ela é conhecida por ser portátil, orientada a objetos e amplamente utilizada para o desenvolvimento de aplicativos web, aplicativos móveis, sistemas embarcados e muito mais.

JavaScript

Linguagem de programação de alto nível, interpretada e orientada a objetos, amplamente utilizada para criar conteúdo interativo em páginas da web. Ela é executada no lado do cliente (navegador) e permite adicionar funcionalidades dinâmicas, como animações, validações de formulários e interações com o usuário. JavaScript é uma das tecnologias fundamentais da web moderna e é essencial para o desenvolvimento de aplicativos web dinâmicos e responsivos. Sua sintaxe é influenciada pela linguagem de programação Java, mas JavaScript é uma linguagem independente com suas próprias características e funcionalidades.
